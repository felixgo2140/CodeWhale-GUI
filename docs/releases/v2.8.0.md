# CodeWhale GUI v2.8.0

这是一次面向日常稳定性与新用户安装的完整发布。它整合了近期的模型升级、语音与附件输入、多模型对比、长任务恢复、Harness 产物交付，以及一套真正执行安装的发布验证流程。

## 主要更新

- 新增 GPT 5.6 Fable / Sol / Luna、Claude Fable 5 与受保护的 Qwen 3.8 Max Preview Token Plan 配置。
- 图片附件先完成本地 OCR 再进入发送队列，远程识别在后台补强，不再让文字先跑、截图后到。
- 语音输入补齐原生权限、稳定回填、取消和强制解锁；支持 `Command+D` 开始/结束录音、`Esc` 停止当前任务。
- 多模型对比新增纵向排列与最新答案汇总，保留 Markdown 阅读、复制全部回复和引用上下文。
- 长任务在上下文临界前主动压缩，并从运行时与报告目录回收 Markdown/PDF；缺失最终回复时提供明确恢复入口。
- DeerFlow、GPT Researcher、Open Deep Research 与 STORM 支持 Tavily key 池轮换，额度失败不会让整个研究链路立即中断。

## 全新用户安装

- 安装器固定安装兼容的 CodeWhale CLI 0.9.0 到用户目录，不使用 `sudo` 或全局 npm。
- 重装保留模型配置、MCP、插件、token 与登录状态；MCP 只合并 CodeWhale 自带项，不覆盖用户已有 server。
- 原生 `CodeWhale.app` 同时支持 Apple Silicon 与 Intel。
- 发布前会在临时空白 HOME 中执行一次完整安装，检查 payload、文件权限、配置保留、launchd plist、通用架构、签名和 GUI 启动；任一项失败都不会发布。

### 安装步骤

1. 安装 Node.js LTS 与 Python 3。
2. 下载 `codewhale-installer.tar.gz` 并解压。
3. 双击 `install.command`；首次被 Gatekeeper 拦截时，右键它选择「打开」。
4. CodeWhale 自动安装并启动。首次使用时到「设置 → 大模型」配置自己的 API key 或 OAuth。

完整安装包不包含任何用户 API key、OAuth 凭据、会话数据或私有插件。
