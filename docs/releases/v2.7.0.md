# CodeWhale GUI v2.7.0

这是一次正式整合发布：将此前分散完成的 GUI 重构、多模型会话稳定性、插件/skills、研究 harness、文件预览、模型路由、安全修复和 macOS 通知统一打入一个可升级、可全新安装的版本。

## 重点更新

- **更清楚的桌面界面**：模块化前端、强化区域对比度、稳定的选中态与更紧凑的工作区。
- **多模型会话可持续**：按唯一 comparison session 归组，标签页切换，模型关闭再打开仍恢复原历史；长会话按需加载，显著降低 CPU 占用。
- **Cron Jobs 与置顶**：Cron Jobs 独立标签排在置顶之前，普通对话和多模型会话都可标记。
- **macOS 完成通知**：每轮完成、失败或中断都会弹原生通知，持久化去重，不补弹历史任务。
- **时间线与智能命名**：首条消息后生成一次目的型短标题；输入时间线可直接跳回长对话中的对应位置。
- **研究与文件交付**：Markdown 报告直接显示，PDF/文档附件以文件卡交付，可预览、下载、用系统 App 打开或在 Finder 中显示。
- **插件与 skills**：本地目录/GitHub 安装、启停、修复和更新检查统一管理；子 skill 从 `+` 菜单直接调用。
- **研究 harness 扩展**：随包提供 DeerFlow、GPT Researcher、Open Deep Research、STORM、Agent Loop、Pydantic AI、CrewAI、Browser Use 和 Obsidian 等无密钥脚本与桥接。
- **CodeWhale v0.9 兼容**：按真实 thread provider/model 恢复会话，支持 ChatGPT OAuth、Kimi K3 订阅路由和多种 OpenAI-compatible provider。
- **安全加固**：更新包 Ed25519 签名 + SHA-256 校验；加强凭据下载、命令/路径校验、安全解包、插件来源与 harness 防回退。

## 下载与安装

### 新用户

下载 **`codewhale-installer.tar.gz`**，解压后双击 `install.command`。安装包包含 GUI、原生 macOS App、安装器、Claude 订阅桥接二进制和研究 harness 脚本；API key、OAuth 凭据和用户会话不会打入安装包。

### 已安装用户

在 CodeWhale 的「设置 → 更新」中检查 GUI 更新。客户端会先验证 `manifest.json.sig`，再校验各资产 SHA-256，验证通过后才替换文件。

## 资产说明

- `codewhale-installer.tar.gz`：全新安装包。
- `gui-2.7.0.tar.gz`：GUI 在线更新包。
- `harness-2.7.0.tar.gz`：研究 harness 在线更新包。
- `manifest.json` / `manifest.json.sig`：签名更新清单。
- `CodeWhale.app.tar.gz`：原生 macOS 壳更新包。
- `codewhale-claude` / `codewhale-tui`：Claude 订阅兼容运行时。
- `SHA256SUMS`：公开下载校验值。

## 升级说明

- 当前用户配置、模型 key、OAuth 文件、插件、会话、归档和研究输出均保留。
- harness 包不含任何密钥；只有本机已存在 `~/agent-harnesses/harness.env` 时才会安装需要密钥的外部引擎。
- 从很早的版本升级后，建议退出并重新打开一次 CodeWhale，让原生 App 和所有 sidecar 使用新版本。
