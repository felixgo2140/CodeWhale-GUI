# CodeWhale GUI v2.7.3

这是 v2.7.2 的附件输入可靠性热修复。它解决图片或文件仍在上传、落盘时,用户按回车后文字消失而附件继续挂在输入区的问题。

## 修复内容

- 补齐附件发送函数的 ES module 导出,消除输入框清空后才出现的异步未定义错误。
- 单模型发送失败时恢复原文字和附件,用户可以直接重试。
- 任务运行中的排队消息同样保留草稿与附件,并继续严格按输入顺序发送。
- 多模型全体发送和单栏追问采用相同恢复规则,不会因某一栏附件准备失败而吞掉输入。
- 新增模块导出和失败恢复测试,覆盖这次故障的直接根因。

## 下载与升级

### 已安装用户

在 CodeWhale 的「设置 → 更新」检查 v2.7.3。客户端会验证 Ed25519 签名和 SHA-256 后完成更新。刷新界面即可加载修复;原生 App 更新后退出并重新打开一次可确保全部资产一致。

### 新用户

下载 `codewhale-installer.tar.gz`,解压后双击 `install.command`。完整安装包不包含用户 API key、OAuth 凭据、会话数据或私有插件。

## 资产说明

- `codewhale-installer.tar.gz`:完整安装包。
