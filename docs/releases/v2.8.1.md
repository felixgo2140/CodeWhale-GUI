# CodeWhale GUI v2.8.1

这是 v2.8.0 的语音输入稳定性热修复。

## 修复内容

- macOS 语音识别服务短暂中断时自动恢复一次。
- 已经识别出的部分文字会直接交付并进入 Prompt 整理,不会被错误回调清空。
- 自动恢复失败时保留输入框内容并恢复可输入状态。
- 识别中断的错误域和错误码会写入本地日志,不包含口述内容。

## 安装

下载 `codewhale-installer.tar.gz`,解压后双击 `install.command`。安装器会保留已有模型配置、MCP、插件、登录状态与会话数据。

完整安装包继续包含 CodeWhale CLI、通用 macOS App、GUI、Harness 与所需运行时预热流程,并通过隔离 HOME 安装验证。
