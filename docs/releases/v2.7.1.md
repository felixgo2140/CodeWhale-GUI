# CodeWhale GUI v2.7.1

这是 v2.7.0 的完整功能补发版，重点解决“从 GitHub 更新后本机新功能被旧包覆盖”的问题。语音输入、任务操作菜单、后台生命周期与实时状态修复均已进入签名更新包和完整安装包。

## 重点更新

- **Fn 与按钮语音输入**：CodeWhale 位于前台时，按住 `Fn`/Globe 开始中文语音转写，松开结束；单聊、对比总输入框和各模型追问框也可点击麦克风按钮启停。
- **口语自动整理为 Prompt**：转写结果会和输入框草稿合并，删除口头禅、重复与自我修正，保留数字、专名、限制和不确定性；只回填输入框，不自动发送。
- **原生权限完整打包**：macOS App 已加入麦克风和语音识别权限说明，并随 Release 提供双架构 `CodeWhale.app.tar.gz`。
- **重启不再打断模型任务**：GUI 重启会接管健康的 provider sidecar，不再一刀切终止正在执行的 Kimi、ChatGPT、DeepSeek 等后台。
- **状态显示以 runtime 为准**：侧栏仍可快速读取摘要缓存，但运行中、已完成、模型、provider 和更新时间会由本地实时 turn 状态覆盖。
- **禁止隐藏自动续跑**：任务停在“接下来会执行”或缺少最终总结时，只提供手动「继续执行」/「继续总结产出」按钮，不静默创建 turn，避免循环和重复扣费。
- **任务级右键菜单**：增加 Finder 显示、复制工作目录/会话 ID/深度链接、新任务续接、新 Git worktree 续接和新窗口打开。
- **安装器生命周期修复**：frontend launch agent 使用 `AbandonProcessGroup`，仅重启 GUI 时保留独立 provider 后台；显式重置、更新运行时或配置变更仍会重启对应后端。

## 下载与升级

### 已安装用户

在 CodeWhale 的「设置 → 更新」检查 v2.7.1。客户端会验证 Ed25519 签名和 SHA-256 后再替换 GUI、Harness 与原生 App。升级完成后退出并重新打开一次 CodeWhale，以加载新的原生语音能力。

### 新用户

下载 `codewhale-installer.tar.gz`，解压后双击 `install.command`。完整安装包包含 GUI、原生 App、研究 Harness 和可选运行时，不包含任何用户 API key、OAuth 凭据或会话数据。

## 资产说明
