05这次重写改什么

三类改动，视觉之外的部分随重构一并偿还。

  • 视觉全组件按 token 重做——侧栏 / 消息流 / 工具卡 / 审批卡 / 对比窗 / 模态 / 输入区 / 顶栏，一种视觉语言贯穿；五种脉冲动画合并为一个呼吸点。
  • 结构样式唯一真源——干掉 index.html 里 3146 行内联样式与 styles.css 的双写，compare.css 并入同一 token 体系；JS 里 35 处直接改 style 的地方收敛为 class 切换。
  • 结构逻辑分层梳理——单文件 3309 行拆为清晰模块（API 层 / 消息流 / 对比窗 / 面板），你做过的每个功能先盘点造册，重构后逐一验收，不丢一件。
  • 修复兼容性偿债——color-mix() 换成 macOS 12 WKWebView 可用的写法；重复选择器、死规则清除;动画尊重 prefers-reduced-motion。

确认这版方向后：我出完整设计规范与组件清单 → Codex 按规范实施 → 我逐界面截图验收回归。
